# Testo Helper Changelog
## [2.6.1] - 19.06.2026
- Исправлена ошибочная подсветка слова `image` внутри имён макросов (например, `count_image`)
- Исправлена подсказка при наведении на макрос, если между шапкой комментариев и `macro` стоит объявление `param`
## [2.6.0] - 04.06.2026
- 🔥 Документация при наведении на макросы: блок комментариев над определением макроса показывается в виде оформленной подсказки
## [2.5.0] - 19.02.2026
- 🔥 Добавлено 25 новых сниппетов (итого 27): циклы, условия, клавиатурные комбинации, действия мыши, ожидание
- Новая вкладка "Сниппеты" в панели справки с группировкой по категориям
## [2.4.3] - 30.01.2026
- Форматирование сохраняет оригинальные отступы (табы/пробелы) если уровень вложенности верный, включая файлы со смешанной табуляцией
## [2.4.2] - 28.01.2026
- Добавлена документация для `snapshot`: управление временными снимками ВМ
- Новая категория "Переменные": динамические переменные `$<VAR>`, создание через `testo-guest-additions-cli`
- Добавлена документация для `step`: маркер шага теста с автоинкрементом
- Расширена документация для `if`: все типы выражений, операторы сравнения строк и чисел, `else if`
- Расширена документация для `mouse`: все действия, типы destination, спецификаторы позиционирования
## [2.4.1] - 28.01.2026
- Расширена документация для `exec`: добавлены параметры `with`, `as`, `expect`
## [2.4.0] - 28.01.2026
- 🔥 Новая вкладка "Клавиши для press" в панели справки
  - Полный список всех клавиш для команды `press`, сгруппированных по категориям
  - Клавиши кликабельны — нажатие копирует название в буфер обмена
- Добавлена ссылка "Список всех клавиш" в подсказке при наведении на `press`
## [2.3.1] - 28.01.2026
- Добавлен пункт контекстного меню "Testo Helper: Справка" (правый клик в редакторе .testo файла)
## [2.3.0] - 27.01.2026
- 🔥 Новый функционал: панель справки по командам Testo
  - Команда `Testo Helper: Справка` (`Ctrl+Shift+P` → "Testo Helper: Справка")
  - Поиск по функциям (название, синтаксис, описание)
  - Фильтрация по категориям: Действия с ВМ, Управление ВМ, Клавиши, Устройства, Копирование, Объявления, Управление потоком, Изображения
  - Поиск и категория работают совместно
- 🔥 Новый функционал: документация при наведении на встроенные функции
- Добавлена настройка `Enable Docs Hover` для включения/отключения документации
- Документация вынесена в отдельный файл `builtinDocs.js` для удобства расширения и актуализации
